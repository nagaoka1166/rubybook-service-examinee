# Tesma (アプリケーション名)

## サービス概要
Tesmaとは, 被験者を集めたい立命館大学の研究者と立命館大学の学生をマッチングさせるサービスです。


## 開発環境

### ・ 使用技術  
      ・ Ruby 2.7.3  
      ・ Rails 6.1.4.1  
      ・ bootstrap5  
      ・ Javascript
      ・ CSS(SCSS)
      ・ Haml
            
### ・ 主なgem  
      ・ devise
      ・ enum
      ・ kaminari
      ・ pry-rails
      ・　ransack
      ・ sidekiq
      ・ font-awesome-sass

### ・ ストレージサービス  
      ・ Amazon S3
