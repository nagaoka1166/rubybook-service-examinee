# Tesma
サービス名は、tester(被験者)をmatching(マッチング）させるというサービスの目的から`Tesma`にいたしました。

## サービス概要
Tesmaとは, 被験者を必要とする立命館大学の研究者と実験に興味がある立命館大学の学生をマッチングさせるサービスです。
## なぜ作ったか
私が所属する立命館大学スポーツ健康科学部では、被験者を必要とする実験が多く行われています。  
しかし、被験者を集めること
